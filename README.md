# Sistema de Rifa en C++

Este proyecto implementa un sistema de rifa simple utilizando C++.  
Permite asignar nombres a números del 1 al 100, mostrar los participantes,
elegir finalistas aleatorios y seleccionar un ganador final.

## 🚀 Funcionalidades
- Mostrar números disponibles y asignados
- Asignar nombre a un número
- Elegir 5 finalistas aleatorios
- Elegir un ganador final
- Menú interactivo

## 🛠️ Tecnologías
- C++
- std::vector
- Funciones aleatorias (`rand`, `srand`)
- Control de flujo con `do-while`

# C++ Raffle System

This project implements a simple raffle system using C++.  
It allows assigning names to numbers from 1 to 100, displaying participants,
selecting 5 random finalists, and choosing a final winner.

## 🚀 Features
- Display available and assigned numbers
- Assign a name to a number
- Randomly select 5 finalists
- Choose the final winner
- Interactive console menu

## 🛠️ Technologies Used
- C++
- std::vector
- Random functions (`rand`, `srand`)
- Control flow using `do-while`

## 📦 Compilation
```bash
g++ raffle.cpp -o raffle
